# Superstore Sales Dashboard (Excel)

## Project Overview
This project analyzes Superstore sales data to identify key business insights. The dashboard provides a clear view of sales, profit, customer segments, categories, and regional performance. It helps understand which products drive profit, which regions underperform, and overall business trends.

## Key Features
- Interactive dashboard with slicers (Category, Region, Segment, Year)
- Sales, Profit, and Quantity KPIs
- Monthly Sales Trend Chart
- Profit by Category & Sub-Category
- Top 10 and Bottom 10 Product Analysis
- Region-wise Sales vs Profit comparison
- Return Orders Overview (if included)

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Basic Calculations & Formulas
- Conditional Formatting 
To demonstrate beginner-level to intermediate Excel data analysis skills through cleaning, organizing, visualizing, and interpreting business data. This project helps showcase dashboard creation skills for data analytics job applications.
