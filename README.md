Sales Performance Dashboard – README
Project Purpose

This project analyzes a company’s sales data to understand trends, top-performing products, regional performance, and customer behavior. The purpose is to turn raw data into useful insights that can help a business make better decisions.

Why This Project

To practice real-world data analysis skills used by data analysts.

To show your ability to clean data, build dashboards, and explain insights.

To include a strong, practical Excel project in your resume and portfolio.

What the Project Includes

Data cleaning: removed duplicates, fixed formatting, corrected dates.

Data transformation: used formulas such as VLOOKUP, IF, SUMIF, COUNTIF, TEXT, and pivot tables.

Visual analysis: built charts for sales trends, product performance, monthly revenue, and region-wise comparison.

Interactive dashboard: slicers for month, category, and region to filter data quickly.

How It Works

Raw sales data is cleaned and structured.

Pivot tables summarize key metrics like total sales, quantity sold, top customers, and top products.

Charts visualize trends and make insights easy to understand.

A dashboard links everything together so the user can monitor performance with one click.

Key Insights (Example)

Highest sales come from the Electronics category.

The North region performs better than South and West.

Sales peak in November and December.

Outcome

This dashboard makes it easy for managers to track sales performance, identify growth opportunities, and make data-driven decisions.
